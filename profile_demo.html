📝 Kode yang Dimodifikasi:
1. views.py - my_profile function:
@login_required
def my_profile(request):
    context = {
        'user': request.user,
    }
    return render(request, 'my-profile.html', context)
2. Template variables yang digunakan:
  • {{ user.username }} - Username
  • {{ user.email }} - Email
  • {{ user.get_full_name }} - Nama lengkap
  • {{ user.date_joined|date:"d M Y" }} - Tanggal bergabung
  • {{ user.last_login|date:"d M Y H:i" }} - Login terakhir
  • {{ user.id }} - User ID
  • {{ user.is_active }} - Status aktif
  • {{ user.is_staff }} - Status staff
  • {{ user.is_superuser }} - Status superuser
